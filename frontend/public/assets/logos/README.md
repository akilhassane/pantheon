# Application Logos

This directory contains the application logos for both light and dark themes.

## Files

- **logo-light.jpg** - Logo for light theme (original)
  - White background with black/colored elements
  - Size: 750x750 pixels
  
- **logo-dark.jpg** - Logo for dark theme (generated)
  - Dark background (#0F1115) with white/inverted elements
  - Size: 750x750 pixels
  - Generated by inverting colors from logo-light.jpg

## Color Scheme

### Light Theme
- Background: White (#FFFFFF)
- Foreground: Black and colors

### Dark Theme
- Background: Dark (#0F1115)
- Foreground: White (#FFFFFF) and inverted colors

## Usage

Import the logos in your React components:

```tsx
import Image from 'next/image'

// For light theme
<Image src="/assets/logos/logo-light.jpg" alt="Logo" width={100} height={100} />

// For dark theme
<Image src="/assets/logos/logo-dark.jpg" alt="Logo" width={100} height={100} />
```

Or use the ThemeLogo component for automatic theme switching:

```tsx
import { ThemeLogo } from '@/components/ui/ThemeLogo'

<ThemeLogo width={100} height={100} />
```

## Regenerating Dark Logo

If you need to regenerate the dark theme logo, run:

```powershell
.\create-dark-logo.ps1
```

This script will:
1. Load logo-light.jpg
2. Convert white pixels to #0F1115
3. Convert black pixels to white
4. Invert other colors
5. Save as logo-dark.jpg
